Git es un sistema de control de versiones distribuido, diseñado para manejar proyectos grandes y pequeños de forma eficiente.
Git permite a los desarrolladores trabajar juntos en un proyecto, rastrear los cambios en el código fuente y revertir cambios 
no deseados de manera fácil y rápida. Gitflow es un modelo alternativo de creación de ramas en Git en el que se utilizan ramas 
de función y varias ramas principales.
